# Getting Started

Install Node.js & npm from [https://nodejs.org/en/download/](https://nodejs.org/en/download/).

Then run the following:
```
git clone https://github.com/theSaeed/tic-tac-toe.git
cd tic-tac-toe/
npm install
npm start
```
